# Simple Shooting Game in Modern C++

## Description
This is a basic shooting game written in modern C++, designed for beginners and intermediate learners.
The player controls a character that can move left and right and shoot bullets at falling targets. The goal is to score as many points as possible before the target reaches the player.

## How to Play
- Move the player left using `a` or `A`.
- Move the player right using `d` or `D`.
- Shoot a bullet using `s` or `S`.
- Each time a bullet hits a target, your score increases by 1.
- The game ends if a target reaches your player.

## Features
- Console-based game using standard C++ libraries.
- Random target generation for replayability.
- Score tracking and display.
- Easy to run on any modern C++ compiler.

## Tools Used
- C++ Standard Library (`<iostream>`, `<cstdlib>`, `<ctime>`, `<thread>`).
- Random numbers for target positions.
- Thread sleep to control game speed.

## Conclusion
This project demonstrates basic game logic, input handling, and randomization in C++.
It’s a fun way to learn loops, conditionals, and console I/O.
